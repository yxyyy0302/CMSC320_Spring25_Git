Q1: Why might we consider using Git?

Answer: Because it's a powerful version control system which allows us to add, make changes to, and delete 
files. It's also great for code collaboration. 

Q2: What field could data science be applied to? Give an Example.

Answer: A field that data science could be applied to is healthcare, because it can be used to analyze very
large datasets of patient information and health status. With the help of data science, patients can be
treated more effectively and efficiently.
